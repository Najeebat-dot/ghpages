Summary of Seminar 1

Computer architecture refers to the design and organization of a computer system, including its hardware and software components. The architecture defines how these components interact with each other to perform various functions. This article provides an overview of the basic components of a computer, their structure, and their functionality.

Central Processing Unit (CPU):

The CPU is often considered the "brain" of the computer. It executes instructions and performs calculations. The CPU consists of the following components:

- Control Unit: It manages the execution of instructions, coordinates data movement, and controls the flow of information within the CPU.

- Arithmetic Logic Unit (ALU): It performs arithmetic and logical operations, such as addition, subtraction, and comparison.

- Registers: These are small, high-speed memory units that store data and instructions that the CPU needs to access quickly.

Memory:

Memory stores data and instructions that the CPU can access. The primary types of memory include:

- Random Access Memory (RAM): It provides temporary storage for data and instructions that are actively used by the CPU. RAM is volatile, meaning its contents are lost when the computer is powered off.

- Read-Only Memory (ROM): It contains permanent instructions or data that are stored during manufacturing and cannot be modified by the user.

- Cache: This high-speed memory is closer to the CPU than RAM and holds frequently accessed data and instructions, enabling faster processing.

3. Input Devices:

Input devices allow users to provide data or commands to the computer. Some common input devices include:

- Keyboard: It allows users to input text and commands.

- Mouse: It enables users to control the cursor on the screen and select objects.

- Touchscreen: It senses touch input, allowing users to interact directly with the display.

- Microphone: It captures audio input, such as voice commands or recordings.

4. Output Devices:

Output devices present information or results to the user. Common output devices include:

- Monitor/Display: It presents visual information, including text, images, and videos.

- Printer: It produces hard copies of documents or images.

- Speakers: They generate audio output, allowing users to hear sounds, music, or speech.

5. Storage Devices:

Storage devices are used to store data and programs for long-term use. They include:

- Hard Disk Drive (HDD): It provides large-capacity storage for programs and data using magnetic storage.

- Solid-State Drive (SSD): It offers faster and more reliable storage using flash memory chips.

- Optical Drives: These include CD, DVD, and Blu-ray drives, used to read and write optical discs.

6. Bus System:

The bus system connects different components of the computer, allowing them to communicate and exchange data. It consists of:

- Data Bus: It transfers data between the CPU, memory, and other devices.

- Address Bus: It carries the memory addresses of data being read from or written to.

- Control Bus: It carries control signals to coordinate the activities of various components.

7. Operating System (OS):

The operating system is a software component that manages the computer's resources and provides a user interface. Key functions of an OS include:

- Process Management: It manages the execution of programs or processes, allocating resources and scheduling tasks.

- Memory Management: It controls and organizes the allocation of memory resources to different processes.

- File System Management: It provides a hierarchical structure for organizing and accessing files on storage devices.

- Device Management: It controls communication and interaction with input/output devices.

In summary, the architecture of a computer involves the organization and design of its components, which work together to perform various functions. The CPU, memory, input/output devices, storage devices, bus system, and operating system form the fundamental elements of a computer system. Understandingthe structure and functionality of these components is crucial for comprehending how a computer operates and facilitates the execution of tasks and applications.
